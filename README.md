# Kurnex GitHub Pages Website

This repository contains the official GitHub Pages website for Kurnex.

## Files

- `index.html` - Home page
- `privacy-policy.html` - Privacy Policy for Google Play
- `terms-of-service.html` - Terms of Service
- `about.html` - About page
- `styles.css` - Website styling

## Google Play Privacy Policy URL

After enabling GitHub Pages, use this URL format in Google Play Console:

`https://kurnextech.github.io/kurnex/privacy-policy.html`
